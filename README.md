安装 Install
--- 
#### 手动安装 Manually
```
将`PFKit`文件夹拖入到工程目录中即可使用
```

#### CocoaPods
```
platform:ios, '8.0'
pod 'PFKitSwift', '~> 0.0.1'
```

使用 Use
---
#### CocoaPods
```
import PFKitSwift
```
